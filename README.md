# 완독 랜덤 질문

독서동호회 완독에서 모임 중 사용할 수 있는 랜덤 질문 뽑기 웹앱입니다.

## 기능

- 문학 / 소설, 비문학, 에세이 장르 선택
- 공통 질문과 장르별 질문을 섞어서 랜덤 질문 표시
- 같은 질문이 바로 반복되지 않도록 처리
- 질문 복사
- 오늘 뽑은 질문 기록
- 모바일 화면 대응
